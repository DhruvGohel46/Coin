# flip a Coin (BIOSORA Inspired)

A high-fidelity, 3D decision-making engine built with Next.js, Framer Motion, and Tailwind CSS.

## 🚀 Deployment to Vercel

This project is optimized and ready for deployment on Vercel.

### Option 1: Vercel CLI
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the root directory.
3. Follow the prompts.

### Option 2: GitHub Integration (Recommended)
1. Push this repository to GitHub.
2. Connect the repository to Vercel via the [Vercel Dashboard](https://vercel.com/new).
3. Vercel will automatically detect Next.js and build the project.

## ✨ Features
- **3D Physics-like Coin**: Rose Gold & Chrome metallic flip animations.
- **Biosora UX**: Tactile dial-slider interface for mobile-first statistics.
- **Premium Design**: Liquid glass effects, earthy palettes, and grain texture overlays.
- **Optimized**: Full TypeScript support and SEO integration.

## 🛠 Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Ionicons (via react-icons)
